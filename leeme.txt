## 🚀 EJECUCIÓN DEL PROYECTO

### Para desarrollar:
1. ng serve
   (Inicia servidor en http://localhost:4200)

### Para compilar:
1. ng build
   (Compila el proyecto con SSR)

### Para terminar:
   Presionar Ctrl + C (back y front)

git branch

git checkout main

git checkout feature/layout-general
