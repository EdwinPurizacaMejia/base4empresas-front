## 🔧 CÓMO INTEGRAR EN TUS COMPONENTES

### 1. Importar servicio y componentes:
import { NotificationService } from '../../services/notification.service';
import { LoadingSpinnerComponent } from '../shared/loading-spinner.component';
import { EmptyStateComponent } from '../shared/empty-state.component';
import { ErrorStateComponent } from '../shared/error-state.component';

### 2. Inyectar servicio:
constructor(
  private service: MyService,
  private notificationService: NotificationService
) {}

### 3. En el template:
<!-- Loading -->
<app-loading-spinner *ngIf="loading && items.length === 0"></app-loading-spinner>

<!-- Error -->
<app-error-state *ngIf="error" [message]="error" (onRetry)="load()"></app-error-state>

<!-- Empty -->
<app-empty-state *ngIf="!loading && items.length === 0" [showAction]="true" (onAction)="onCreate()"></app-empty-state>

<!-- Data -->
<app-table *ngIf="items.length > 0" [data]="items"></app-table>

### 4. En métodos:
this.notificationService.success('¡Listo!');
this.notificationService.error('Error al cargar');

Ver PATRONES_COMUNES.md para 10+ ejemplos listos para copiar/pegar.

## 🔄 FLUJO TÍPICO (Patrón Recomendado)

1. Cargando (loading spinner)
   ↓
2. Datos vacíos? (empty state) | Error? (error state) | Éxito? (mostrar datos)
   ↓
3. Usuario interactúa (crea, edita, elimina)
   ↓
4. Notificación de éxito/error
   ↓
5. Recarga datos (vuelve a paso 1)
